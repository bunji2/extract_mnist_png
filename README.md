# extract_mnist_png
This tool extracts mnist-dataset to png.

# Usage:

```
extract_mnist_png [ cols rows ]
```

# Example:

```
extract_mnist_png
```

to pack 40x20 images.

![40x20](sample/40x20.png)

```
extract_mnist_png 80 80
```

to pack 80x80 images.

![80x80](sample/80x80.png)

```
extract_mnist_png 1 1
```

to save each images.

![1x1](sample/1x1.png)
